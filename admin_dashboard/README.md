# 🚀 Super Admin Dashboard - Ultimate Edition

**LINE Bot Management System** พัฒนาด้วย PySide6

## 🔥 NEW Powerful Features

1. **Firebase/Firestore Integration** - เชื่อมต่อโดยตรง
2. **Real-time Analytics** - กราฟสถิติสด
3. **Quick Reply Builder** - สร้าง Quick Reply ง่ายๆ
4. **Image/Sticker Sender** - ส่งรูปและสติกเกอร์
5. **Webhook Tester** - ทดสอบ Webhook
6. **Schedule Messages** - ตั้งเวลาส่งข้อความ
7. **User Segmentation** - แบ่งกลุ่มผู้ใช้
8. **Export/Import** - นำเข้า-ส่งออกข้อมูล
9. **Error Recovery** - ระบบกู้คืนอัตโนมัติ
10. **Advanced Logging** - บันทึก Log ละเอียด

## 📦 Installation

```bash
pip install -r requirements.txt
python super_admin_dashboard.py
```

## 🔧 Configuration

Create `config.json`:
```json
{
    "LINE_CHANNEL_ACCESS_TOKEN": "YOUR_TOKEN",
    "SUPER_ADMIN_USER_ID": "YOUR_USER_ID"
}
```
