## 📝 ขั้นตอนการเพิ่มความรู้

### Step 1: เปิดฟอร์ม
```
พิมพ์: เพิ่มความรู้
```

### Step 2: คลิก "📋 ดูตัวอย่าง"
- ระบบแสดง Carousel 3 ตัวอย่าง

### Step 3: เลือกตัวอย่างที่ใกล้เคียง
- เลื่อนดูตัวอย่างทั้ง 3
- คลิกปุ่ม "📋 Copy" ในตัวอย่างที่ต้องการ

### Step 4: แก้ไขข้อความ
- Paste ข้อความที่ Copy มา
- แก้ไขเนื้อหาให้ตรงกับความรู้ที่ต้องการเพิ่ม

### Step 5: ส่งข้อความ
- ส่งข้อความในรูปแบบที่กำหนด
- ระบบจะบันทึกเข้า Knowledge Base

### Step 6: รับการยืนยัน
```
✅ เพิ่มความรู้สำเร็จ!

📊 รายละเอียด:
• ID: xyz789
• หมวด: วิธีแก้จริง
• วัสดุ: ABS
• สถานะ: รอตรวจสอบ

💡 ความรู้นี้จะถูกใช้โดย AI ทันที
```

## 🏷️ หมวดหมู่ที่รองรับ

### 1. 🔧 วิธีแก้จริง (real_world_solutions)
**ใช้เมื่อ:** แก้ปัญหาเฉพาะที่เจอจริง ๆ
```
ตัวอย่าง:
- ABS มีรอยยุบ → เพิ่มความดัน Holding
- PP ฉีดไม่เต็ม → เพิ่ม Injection Speed
```

### 2. 📊 พารามิเตอร์ (proven_parameters)
**ใช้เมื่อ:** ค่าพารามิเตอร์ที่ได้ผลจริง
```
ตัวอย่าง:
- ABS ชิ้นงานกล่อง: Temp 220°C, Pressure 80 bar
- PP ฉีดฝา: Temp 200°C, Holding 5 วินาที
```

### 3. 🏭 เฉพาะเครื่อง (machine_specific)
**ใช้เมื่อ:** ปัญหาเฉพาะเครื่องจักรรุ่นหนึ่ง
```
ตัวอย่าง:
- Haitian MA900 มีเสียงดัง → ตรวจ Hydraulic Oil
- Engel Victory 200 ไม่ปิดแม่พิมพ์ → ตรวจ Sensor
```

### 4. 💡 เคล็ดลับ (expert_tips)
**ใช้เมื่อ:** เทคนิคพิเศษหรือ shortcuts
```
ตัวอย่าง:
- อบเม็ดประหยัดไฟ → ตากแดด 2-3 ชั่วโมง
- ทดสอบความชื้น → เผาเม็ดพลาสติก 1 เม็ด
```

### 5. 📖 คำศัพท์ (local_terminology)
**ใช้เมื่อ:** แปลคำศัพท์หรืออธิบายคำ
```
ตัวอย่าง:
- "ครีบ" คือ Flash - พลาสติกล้นออกจากแม่พิมพ์
- "รอยยุบ" คือ Sink Mark - เกิดจากหดตัวไม่สม่ำเสมอ
```

### 6. 📚 กรณีศึกษา (case_studies)
**ใช้เมื่อ:** เคสจริงจากลูกค้า
```
ตัวอย่าง:
- ลูกค้า ABC ฉีดกล่อง PP Short Shot → แก้ด้วยเพิ่ม Gate + เพิ่ม Temp 15°C
- โรงงาน XYZ ชิ้นงาน PC บิดงอ → แก้ด้วย Cooling System ใหม่
```

### 7. 🧪 วัสดุท้องถิ่น (local_materials)
**ใช้เมื่อ:** ข้อมูลวัสดุจากแหล่งท้องถิ่น
```
ตัวอย่าง:
- PP ไทย vs PP จีน → MFI ต่างกัน 20%
- ABS ของ PTT vs IRPC → ความหนืดต่างกัน
```

### 8. 🏪 ซัพพลายเออร์ (supplier_info)
**ใช้เมื่อ:** ข้อมูลร้านค้า/ซัพพลายเออร์
```
ตัวอย่าง:
- หาซื้อ ABS ย่านลาดพร้าว → ร้าน XYZ ราคา 65 บาท/กก.
- ซ่อมเครื่องฉีดย่านบางนา → ช่าง ABC โทร 08x-xxx-xxxx
```

## 🎯 Tips สำหรับ Super Admin

### ✅ DO:
- ✅ ใช้ภาษาไทยที่เข้าใจง่าย
- ✅ ระบุตัวเลขที่เจาะจง (เช่น 10-20°C, 5-10 วินาที)
- ✅ ใส่ชื่อวัสดุให้ชัดเจน (ABS, PP, PC)
- ✅ ใช้แท็กที่เกี่ยวข้อง เพื่อค้นหาง่าย
- ✅ อธิบายวิธีแก้ให้ละเอียด ทำได้จริง

### ❌ DON'T:
- ❌ อย่าใส่ข้อมูลสั้นเกินไป
- ❌ อย่าลืมระบุหน่วย (°C, บาท/กก., วินาที)
- ❌ อย่าใช้คำคลุมเครือ ("ลองปรับดู", "อาจจะ")
- ❌ อย่าใส่ข้อมูลที่ไม่แน่ใจ
- ❌ อย่าคัดลอกจากอินเทอร์เน็ตโดยไม่ตรวจสอบ

## 📊 การตรวจสอบความรู้

### Auto-Detection:
ระบบจะตรวจสอบและจัดหมวดอัตโนมัติ:
- ✅ ตรวจสอบความยาวข้อความ
- ✅ Extract keywords (ABS, PP, รอยยุบ, ฉีด, ฯลฯ)
- ✅ ตรวจหาวัสดุที่กล่าวถึง
- ✅ จัดหมวดหมู่ตามบริบท

### Verification Status:
```
pending → ระบบรอตรวจสอบ (ใช้งานได้ทันที)
verified → ผ่านการยืนยัน (ความเชื่อมั่นสูง)
rejected → ไม่ผ่าน (ไม่ใช้ในระบบ)
```

## 📈 Success Metrics

   - ความรู้ทุกรายการมีคุณภาพ
