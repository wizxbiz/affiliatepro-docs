│ 3. ความเร็วฉีดต่ำ                (20%) │
│ 4. อุณหภูมิต่ำ                   (15%) │
│ 5. Venting ไม่ดี                 (5%)  │
└─────────────────────────────────────────┘

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   • เพิ่ม injection pressure 10-20 bar
   • ตรวจสอบ pressure profile

   • เพิ่ม injection speed 10-15%

   • เพิ่ม barrel temp 5-10°C
