🛡️ ความเป็นส่วนตัวของคุณสำคัญ: TukTuk Thailand มุ่งมั่นในการปกป้องข้อมูลส่วนบุคคลของผู้ใช้ทุกท่าน

1. ข้อมูลที่เราเก็บรวบรวม

ประเภทข้อมูล รายละเอียด จัดเก็บ
ข้อมูลจาก LINE User ID, Display Name, Profile Picture ✓
พฤติกรรมการรับชม ประวัติการดูวิดีโอ, การกดถูกใจ, การติดตาม ✓
ข้อมูลการซื้อขาย ประวัติคำสั่งซื้อ, ข้อมูลการจัดส่งสินค้า ✓
ข้อมูลการเงิน รายละเอียดบัตรเครดิตหรือข้อมูลธนาคาร ✗

2. วัตถุประสงค์การใช้ข้อมูล

  • ให้บริการ: เพื่อนำเสนอวิดีโอความบันเทิงและสินค้าที่ตรงกับความสนใจ
  • ตลาดซื้อขาย: เพื่อประมวลผลคำสั่งซื้อและบริการจัดส่งสินค้า
  • การเชื่อมต่อ: เพื่อแจ้งเตือนสถานะการซื้อขายและข่าวสารผ่าน LINE
  • พัฒนาบริการ: วิเคราะห์สถิติเพื่อเพิ่มประสิทธิภาพระบบ Feed และ Marketplace
  • สินค้าแนะนำ: เพื่อแสดงสินค้าและโฆษณาที่เกี่ยวข้องในหน้ากล่องข้อความ โดยอิงจากประวัติการดูสินค้าและหมวดหมู่ที่คุณสนใจ — ไม่มีการอ่านเนื้อหาข้อความส่วนตัวเพื่อวัตถุประสงค์นี้

3. สินค้าแนะนำและโฆษณาในกล่องข้อความ

🔒 ข้อความส่วนตัวของคุณปลอดภัย: แอป TukTuk ไม่มีการอ่าน สแกน หรือประมวลผลเนื้อหาข้อความ เพื่อนำไปใช้ในการแสดงโฆษณาหรือสินค้าแนะนำใด ๆ ทั้งสิ้น

วิธีที่เราแสดงสินค้าแนะนำในกล่องข้อความ:

  • Positional Ads (สุ่มตามตำแหน่ง): การ์ดสินค้าจะถูกแทรกระหว่างรายการสนทนาตามตำแหน่งที่กำหนด เช่น ทุก 5 รายการ โดยดึงจากสินค้าที่ร้านค้าซื้อพื้นที่โฆษณา — ไม่เกี่ยวข้องกับเนื้อหาการสนทนา
  • Interest-based Recommendations (อิงจากประวัติการใช้งาน): สินค้าแนะนำอาจอิงจากหมวดหมู่สินค้าที่คุณเคยดูหรือซื้อใน Marketplace, คะแนนความสนใจ (interest score) ที่บันทึกจากพฤติกรรมการเลื่อนดูสินค้า และจังหวัดหรือพื้นที่จากโปรไฟล์ผู้ใช้
  • Product Chat Context: ในแท็บ "สินค้า" ซึ่งเป็นพื้นที่สำหรับการสื่อสารเรื่องการซื้อขายโดยเฉพาะ อาจมีการแสดงสินค้าที่เกี่ยวข้องกับบทสนทนาปัจจุบัน โดยข้อมูลนี้มาจาก metadata ของสินค้า (เช่น ชื่อสินค้า, หมวดหมู่) ไม่ใช่เนื้อหาข้อความ

การแสดงผลโฆษณา: การ์ดสินค้าแนะนำทุกชิ้นจะมีป้ายกำกับ สินค้าแนะนำ หรือ Sponsored อย่างชัดเจน เพื่อแยกแยะจากรายการสนทนาทั่วไป

การปิดการแสดงสินค้าแนะนำ: คุณสามารถปิดการแสดงสินค้าแนะนำในกล่องข้อความได้ที่ การตั้งค่า → ความเป็นส่วนตัว → สินค้าแนะนำในกล่องข้อความ หรือแจ้งทีมงานผ่านช่องทางการติดต่อในข้อ 8

4. การแบ่งปันข้อมูล

⛔ เราไม่ขาย: ข้อมูลส่วนบุคคลของคุณจะไม่ถูกขายให้บุคคลที่สาม

เราอาจแบ่งปันข้อมูลในกรณีต่อไปนี้เท่านั้น:

  • ผู้ให้บริการโครงสร้างพื้นฐาน: Google Cloud, Firebase, LINE (เป็น Data Processor ตาม DPA)
  • ร้านค้าใน Marketplace: เฉพาะข้อมูลที่จำเป็นสำหรับการดำเนินคำสั่งซื้อ เช่น ชื่อ, ที่อยู่จัดส่ง
  • ตามกฎหมาย: เมื่อมีคำสั่งจากหน่วยงานราชการที่มีอำนาจตามกฎหมาย
  • ความยินยอม: เมื่อคุณให้ความยินยอมโดยชัดแจ้ง
✅ ข้อมูลโฆษณา: ข้อมูลที่ใช้ในการแสดงสินค้าแนะนำ (interest score, หมวดหมู่, จังหวัด) ถูกประมวลผลภายในระบบ TukTuk เท่านั้น ไม่ถูกแบ่งปันให้ Ad Network ภายนอก

5. การรักษาความปลอดภัย

  • 🔐 ข้อมูลถูกเข้ารหัสระหว่างการส่งผ่าน (HTTPS/TLS)
  • ☁️ จัดเก็บบน Firebase Cloud ที่มีมาตรฐาน SOC 2
  • 🛡️ มีการควบคุมการเข้าถึงระดับผู้ใช้ (Firestore Security Rules)
  • 🔒 ข้อความส่วนตัวถูกจัดเก็บด้วย Firestore Security Rules ที่จำกัดการเข้าถึงเฉพาะผู้เกี่ยวข้อง

6. ระยะเวลาการเก็บรักษา

  • ข้อมูลการรับชม / interest score: ตลอดอายุสมาชิกเพื่อแนะนำเนื้อหา — ลบได้เมื่อร้องขอ
  • ข้อมูลคำสั่งซื้อ: จัดเก็บตามระเบียบทางบัญชีและกฎหมายพาณิชย์ (ไม่น้อยกว่า 5 ปี)
  • ข้อความสนทนา: จัดเก็บตลอดไปเพื่อประวัติการซื้อขาย — ลบได้ตามคำขอ
  • Log การใช้งาน: 1 ปี เพื่อความปลอดภัยของระบบ

7. สิทธิ์ของคุณ (PDPA)

  • ✅ เข้าถึง: ขอดูข้อมูลที่เราเก็บรวบรวมเกี่ยวกับคุณ รวมถึง interest score
  • ✅ แก้ไข: ขอแก้ไขข้อมูลที่ไม่ถูกต้อง
  • ✅ ลบ: ขอลบข้อมูลส่วนบุคคล รวมถึงประวัติการดูสินค้าและ interest score
  • ✅ คัดค้าน: คัดค้านการใช้ข้อมูลเพื่อการตลาดหรือโฆษณา
  • ✅ ถอนความยินยอม: ปิดการแสดงสินค้าแนะนำในกล่องข้อความได้ทุกเมื่อ โดยไปที่ การตั้งค่า → ความเป็นส่วนตัว หรือแจ้งทีมงาน
  • ✅ โอนย้ายข้อมูล: ขอรับข้อมูลของคุณในรูปแบบที่อ่านได้ด้วยเครื่อง (Data Portability)

คำขอใช้สิทธิ์จะได้รับการตอบกลับภายใน 30 วัน ตามมาตรา 40 พ.ร.บ. คุ้มครองข้อมูลส่วนบุคคล พ.ศ. 2562
