📸 วิธีสร้าง og-image.jpg

  1. เปิดหน้านี้ใน Browser (Chrome แนะนำ)
  2. กด F12 เปิด DevTools
  3. กดปุ่ม ... (More) > Run command > พิมพ์ "screenshot"
  4. เลือก "Capture node screenshot"
  5. คลิกที่กล่องสีน้ำเงินด้านบน
  6. บันทึกเป็น og-image.jpg ใน folder images/

หรือใช้เครื่องมือ online เช่น htmlcsstoimage.com
