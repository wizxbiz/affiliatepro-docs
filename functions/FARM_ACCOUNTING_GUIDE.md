## ✨ ฟีเจอร์หลัก

### 📝 1. บันทึกด้วยภาษาธรรมชาติ
ไม่ต้องเรียนรู้รูปแบบพิเศษ พิมพ์เหมือนพูดกับเพื่อน

```
✅ "ขายมะม่วง 50 กก. 1,500 บาท"
✅ "ซื้อปุ๋ย 3 กระสอบ 900"
✅ "ค่าแรงตัดหญ้า 500 บาท"
✅ "ได้เงินขายข้าว 8,000"
```

### 📊 2. สรุปรายงานอัตโนมัติ
พิมพ์คำเดียว ได้รายงานทันที

- `สรุปวันนี้` - ยอดวันนี้
- `สรุปสัปดาห์` - ยอด 7 วัน
- `สรุปเดือน` - ยอดเดือนนี้
- `สรุปปี` - ยอดทั้งปี

### 🏷️ 3. จัดหมวดหมู่อัตโนมัติ
AI เข้าใจว่าคุณพิมพ์อะไร และจัดหมวดให้เอง

### 🔄 4. สลับโหมดได้
- 🌾 **โหมดฟาร์ม** - หมวดเกษตร (ปุ๋ย, ยา, เมล็ดพันธุ์)
- 🏪 **โหมดร้านค้า** - หมวดค้าปลีก (ค่าเช่า, สต๊อก, ค่าไฟ)
