### ตัวอย่างที่ 1: ตรวจสอบสถานะระบบ

**คำถาม:**
```

```

```

**Uptime:** 9.97 วันที่ (กว่า 9.97 ชั่วโมง)
**Node Version:** v20.19.5 (ใช้เวอร์ชันของ Node.js ที่รันบนระบบ)

การใช้งาน Memory ที่รายงานนี้:
  * **heapTotal:** 331.89888 bytes (ขนาดของ Heap ที่จำแพงแล้ว)
  * **heapUsed:** 309.44600 bytes (ขนาดของ Memory ที่ระบบใช้งานจริง)
  * **rss:** 1145.03680 bytes (Resident Set Size - ขนาดของ Memory ที่รันบนระบบ) = ขนาด Memory ให้ทำงาน Code
  * **external:** 37.27439 bytes (Memory ที่ใช้โดย Native Binding ภายนอก)

📊 คำแนะนำ:
* หากค่า "heapUsed" ใกล้เคียงกับ "heapTotal" มากเกินไป อาจารย์สามารถลงมาได้ที่ระบบใช้ Memory ใช้งานสูง
```

**Flex Message:**
- 🟢 Header สีเขียว: "สถานะระบบ"
- ✅ Status: "ปกติ"
- 📊 Progress Bar: แสดง ~93% (สีเขียว)
- 💾 Heap Memory: 295.94 / 317.47 MB
- 📈 RSS Memory: 1095.94 MB
- ⏱️ Uptime: 239.35 ชั่วโมง (14361.3 นาที)

### ตัวอย่างที่ 8: คำนวณอัตราผลตอบแทน ROI

**คำถาม:**
```
คำนวณ ROI ลงทุน 100,000 บาท รายได้ต่อเดือน 8,000 บาท

```

```
**เงินลงทุน:** 100,000 บาท
**รายได้/เดือน:** 8,000 บาท
**รายได้/ปี:** 96,000 บาท

**คืนทุนใน:** 12.5 เดือน (1.04 ปี)
**ROI/ปี:** 96% ต่อปี
```

**Flex Message:**
- 📊 Header: "คำนวณอัตราผลตอบแทนการลงทุน (ROI)"
- 📋 รายละเอียด:
  - เงินลงทุนเริ่มต้น: 100,000 บาท
  - รายได้ต่อเดือน: 8,000 บาท
  - รายได้ต่อปี: 96,000 บาท
  - ระยะเวลาคืนทุน: 12.5 เดือน
  - ROI ต่อปี: 96 %
  - ผลตอบแทนสุทธิ: 96,000 บาท/ปี
- 💡 คำแนะนำ: อัตราผลตอบแทนสูงมาก! เหมาะสำหรับการลงทุน

### ตัวอย่างที่ 10: คำนวณอุณหภูมิ (Celsius/Fahrenheit)

**คำถาม:**
```
แปลง 25 องศาเซลเซียส เป็น ฟาเรนไฮต์
```

**คำตอบ:**
```
การแปลงอุณหภูมิ:

**จาก:** 25°C
**เป็น:** 77°F

**สูตร:** °F = (°C × 9/5) + 32
```

**Flex Message:**
- 📊 Header: "การแปลงหน่วยอุณหภูมิ"
- 📋 รายละเอียด:
  - อุณหภูมิ (Celsius): 25 °C
  - อุณหภูมิ (Fahrenheit): 77 °F
  - อุณหภูมิ (Kelvin): 298.15 K
  - สูตรที่ใช้: (C × 9/5) + 32
- 💡 คำแนะนำ: อุณหภูมิที่เหมาะสมสำหรับการทำงาน

## 💡 Tips สำหรับการใช้งาน

1. **ใช้ภาษาธรรมชาติ:** บอทสามารถเข้าใจคำถามแบบพูดธรรมดา
   - ✅ "คำนวณพื้นที่ 5x10"
   - ✅ "หาปริมาตร กว้าง 5 ยาว 10 สูง 3"
   - ✅ "เช็คสถานะระบบ"

2. **ระบุหน่วยให้ชัดเจน:** ช่วยให้ผลลัพธ์แม่นยำ
   - ✅ "220V 15A"
   - ✅ "5 เมตร x 10 เมตร"

3. **ขอคำแนะนำเพิ่มเติม:** บอทจะให้คำแนะนำที่เป็นประโยชน์
   - ✅ "แนะนำวิธีลดต้นทุน"
   - ✅ "ควรใช้วัสดุอะไรดี"
