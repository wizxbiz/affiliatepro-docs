## 💵 คำสั่งดูกำไร

| คำสั่ง | คำอธิบาย |
|--------|----------|
| `กำไรวันนี้` | ดูกำไรของวันนี้ พร้อมยอดขาย, ต้นทุน, กำไรขั้นต้น, กำไรสุทธิ |
| `กำไรเมื่อวาน` | ดูกำไรของเมื่อวาน |
| `กำไรสัปดาห์` | ดูกำไรสัปดาห์นี้ |
| `กำไรเดือน` | ดูกำไรเดือนนี้ (แนะนำ) |

### 📊 ข้อมูลที่แสดง:
- **💰 ยอดขาย** - รายรับทั้งหมด
- **📦 ต้นทุน** - ค่าซื้อสินค้า/วัตถุดิบ
- **📊 กำไรขั้นต้น** - ยอดขาย - ต้นทุน
- **💸 ค่าใช้จ่าย** - ค่าใช้จ่ายอื่นๆ (ค่าเช่า, ค่าส่ง, etc.)
- **✨ กำไรสุทธิ** - กำไรขั้นต้น - ค่าใช้จ่าย
- **% Margin** - เปอร์เซ็นต์กำไรต่อยอดขาย

## 📱 ตัวอย่างการใช้งาน

### 🌅 เปิดร้านตอนเช้า
```
ขาย 500
→ ✅ บันทึก รายรับ: ขายสินค้า ฿500
```

### 📦 รับของเข้าร้าน
```
ซื้อสต๊อก 2000
→ ✅ บันทึก รายจ่าย: ซื้อสินค้า ฿2,000
```

### 🛒 ขายออนไลน์
```
ขาย shopee 350
→ ✅ บันทึก รายรับ: ขายออนไลน์ ฿350
```

### 🚚 จ่ายค่าส่ง
```
ค่าส่ง 50
→ ✅ บันทึก รายจ่าย: ขนส่ง ฿50
```

### 📊 ดูกำไรตอนปิดร้าน
```
กำไรวันนี้

→ 🏪 Dashboard ร้านค้า - วันนี้
   💰 ยอดขาย: ฿850
   📦 ต้นทุน: ฿2,000
   📊 กำไรขั้นต้น: -฿1,150
   💸 ค่าใช้จ่าย: ฿50
   ⚠️ ขาดทุนสุทธิ: ฿1,200
```

## 🎯 Tips & Tricks

### ✨ เคล็ดลับการใช้งาน

1. **บันทึกทันที** - พิมพ์ทันทีเมื่อมีรายการ ไม่ต้องรอสิ้นวัน
2. **ระบุช่องทาง** - ใส่ชื่อช่องทางเพื่อวิเคราะห์ได้ง่าย เช่น `ขาย shopee 300`
3. **ดูกำไรบ่อยๆ** - เช็ค `กำไรวันนี้` ก่อนปิดร้านทุกวัน
4. **แยกต้นทุน** - ใช้ `ซื้อสต๊อก` สำหรับสินค้า และ `จ่าย` สำหรับค่าใช้จ่ายอื่น

### ⚠️ ข้อควรระวัง

- ต้นทุนสินค้า (ซื้อสต๊อก) จะถูกแยกจากค่าใช้จ่ายอื่นๆ
- กำไรขั้นต้น = ยอดขาย - ต้นทุนสินค้า
- กำไรสุทธิ = กำไรขั้นต้น - ค่าใช้จ่ายอื่นๆ
